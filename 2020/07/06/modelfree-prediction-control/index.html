Model-free Prediction

前文讨论的MDP实际上是model-based的，相对于无模型（Model-free）概念，model-based也就是说MDP中的R、P\mathcal{R}、\mathcal{P}R、P，全局的奖励函数和状态转移函数是智能体能直接获取的。

对于Model-free的MDP来说，智能体只有要通过和环境进行交互才能得到环境的反馈。一次交互获得一组反馈，并非全局的奖励函数和状态转移函数。正如下图，

对于Model-based的智能体，可以用经验采样的方式，simulation许多的路径(trajectories)从而评估策略的好坏。

蒙特卡洛策略评估

​ 蒙特卡洛是典型的经验采样方式之一，从初始状态sss开始采样直到终止状态，整个序列作为路径，再累加路径的奖励值Gt=Rt+1+γRt+2+γ2Rt+3+…G_{t}=R_{t+1}+\gamma R_{t+2}+\gamma^{2} R_{t+3}+\ldotsGt​=Rt+1​+γRt+2​+γ2Rt+3​+…，从而求得经验平均奖励(empirical mean return)，也就是值函数vπ(s)=Eτ∼π[Gt∣st=s]v^{\pi}(s)=\mathbb{E}_{\tau \sim \pi}\left[G_{t} \mid s_{t}=s\right]vπ(s)=Eτ∼π​[Gt​∣st​=s]

​ 可以看出，蒙特卡洛方法只适用于episodic MDPs，也就是有终止状态的马尔可夫决策过程。

​ 形式化表示蒙特卡洛策略评估的流程如下：

  1. 从初始状态sss出发直到终止状态，作为一个采样的路径，时间步ttt表示sss在路径上的位置
  2. 记录样本个数N(s)←N(s)+1N(s) \leftarrow N(s) + 1N(s)←N(s)+1
  3. 记录所有样本的奖励值总和A(s)←A(s)+GtA(s) \leftarrow A(s) + G_tA(s)←A(s)+Gt​
  4. 循环1-3步直到采样了大量路径
  5. 得到经验平均奖励v(s)=A(s)/N(s)v(s) = A(s) / N(s)v(s)=A(s)/N(s)

​ 根据大数定律，当采样数趋于无穷大时N(s)→∞N(s) \rightarrow \infinN(s)→∞，经验平均奖励v(s)v(s)v(s)就等于真实值函数vπ(s)v^\pi(s)vπ(s)。

​ 对于蒙特卡洛采样，一般可以使用增量方式来计算经验平均奖励，设其中x1,x2,…x_1,x_2, \dotsx1​,x2​,…表示不同采样的累计奖励，vnv_nvn​表示nnn条路径的平均奖励，则有推导：

vn=1n∑j=1nxj=1n(xn+∑j=1n−1xj)=1n(xn+(n−1)vn−1)=vn−1+1n(xn−vn−1)\begin{aligned} v_{n} &=\frac{1}{n} \sum_{j=1}^{n} x_{j} \\ &=\frac{1}{n}\left(x_{n}+\sum_{j=1}^{n-1} x_{j}\right) \\ &=\frac{1}{n}\left(x_{n}+(n-1) v_{n-1}\right) \\ &=v_{n-1}+\frac{1}{n}\left(x_{n}-v_{n-1}\right) \end{aligned} vn​​=n1​j=1∑n​xj​=n1​(xn​+j=1∑n−1​xj​)=n1​(xn​+(n−1)vn−1​)=vn−1​+n1​(xn​−vn−1​)​

​ 通过这样增量式的计算，最直接的好处就是不用每次迭代都要O(t)O(t)O(t)复杂度的求平均，而只需O(1)O(1)O(1)的复杂度。并且可以就改写了上面流程中的第五步的公式为：

v(St)←v(St)+1N(St)(Gt−v(St))v\left(S_{t}\right) \leftarrow v\left(S_{t}\right)+\frac{1}{N\left(S_{t}\right)}\left(G_{t}-v\left(S_{t}\right)\right) v(St​)←v(St​)+N(St​)1​(Gt​−v(St​))

​ 此外这种类似v(St)←v(St)+α(Gt−v(St))v\left(S_{t}\right) \leftarrow v\left(S_{t}\right)+\alpha\left(G_{t}-v\left(S_{t}\right)\right)v(St​)←v(St​)+α(Gt​−v(St​))的公式，也能更好地和后面的时间差分TD公式进行比较，便于理解。

时间差分策略评估

相对于蒙特卡洛，时间差分方法采用类似bootstrapping的采样，只采样固定长度的路径，因此能够应用于Continuing MDPs上，也就是没有终止状态的马尔可夫决策过程。

最简单的TD是TD(0)，也就是采样只有一次状态转移的路径，同理可得TD(k)的含义。而TD(∞\infin∞)就是蒙特卡洛。

这里以TD(0)为例，介绍时间差分算法：

  • 从初始状态StS_tSt​出发采样一次状态转移，并用增量形式更新值函数：

Gt+1=Rt+1+γv(St+1)v(St)←v(St)+α(Rt+1+γv(St+1)−v(St))\begin{aligned} G_{t+1} &= R_{t+1} + \gamma v(S_{t+1})\\ v(S_t) & \leftarrow v(S_{t}) + \alpha(R_{t+1} + \gamma v(S_{t+1}) - v(S_t)) \end{aligned} Gt+1​v(St​)​=Rt+1​+γv(St+1​)←v(St​)+α(Rt+1​+γv(St+1​)−v(St​))​

  • TD方法中有两个重要元素

    • TD target：Rt+1+γv(St+1)R_{t+1} + \gamma v(S_{t+1})Rt+1​+γv(St+1​)
    • TD error：Rt+1+γv(St+1)−v(St)R_{t+1} + \gamma v(S_{t+1}) - v(S_t)Rt+1​+γv(St+1​)−v(St​)

  • 如果采样多步，即TD(kkk)其中k>0k>0k>0，则公式变为了：

    v(St)←v(St)+α(Rt+1+⋯+Rt+k+1+γv(St+k+1)−v(St))v(S_t) \leftarrow v(S_{t}) + \alpha(R_{t+1} + \dots + R_{t+k+1} + \gamma v(S_{t+k+1}) - v(S_t)) v(St​)←v(St​)+α(Rt+1​+⋯+Rt+k+1​+γv(St+k+1​)−v(St​))

TD和MC方法的比较

  • 采样路径：
    • TD可以采样不完整的路径
    • MC必须采样完整且有终止的路径
  • 应用环境：
    • TD可以在continuing的环境中
    • MC只能在episodic的环境中
  • 效率：
    • TD利用了马尔可夫性质，在马尔可夫环境中更有效
    • MC没有利用马尔可夫性质，在非马尔可夫环境中更有效
  • 滞后性：
    • TD在每一时间步都可以在线地更新
    • MC需要在采样一条完整路径后才能更新，具有滞后性
  • 方法：
    • TD使用了bootstrapping，也进行了sampling
    • MC没有使用bootstrapping，进行了sampling
    • DP没有sampling，只有bootstrapping

目前的问题
